Messenger Generator Pro

Import file Excel/CSV để tự động tạo hàng loạt màn hình Messenger

Import dữ liệu từ file

Cấu trúc file cần thiết:

File phải có các cột sau (thứ tự không quan trọng):

screen_id contact_name avatar_url user_status message_text sender time_display show_time message_type
1 Nguyen Giang https://example.com/avatar.jpg Đang hoạt động Chào bạn! other 14:30 TRUE text
1 Nguyen Giang https://example.com/avatar.jpg Đang hoạt động https://example.com/photo.jpg me FALSE image
2 Mai Anh https://example.com/avatar2.jpg Hoạt động 5 phút trước Hẹn gặp lại nhé me 15:45 TRUE text

Giải thích các cột:

  • screen_id: ⭐ BẮT BUỘC - ID màn hình (các dòng cùng ID sẽ thuộc 1 màn hình)
  • contact_name: ⭐ BẮT BUỘC - Tên người chat
  • avatar_url: 🔧 TÙY CHỌN - Link ảnh avatar (để trống sẽ dùng avatar đẹp mặc định)
  • user_status: 🔧 TÙY CHỌN - Trạng thái (Đang hoạt động, Hoạt động 5 phút trước, v.v.)
  • message_text: ⭐ BẮT BUỘC - Nội dung tin nhắn HOẶC link ảnh
  • sender: ⭐ BẮT BUỘC - me (tin nhắn của tôi) hoặc other (tin nhắn của người kia)
  • time_display: 🔧 TÙY CHỌN - Thời gian hiển thị (VD: 14:30, 12 THG 7)
  • show_time: 🔧 TÙY CHỌN - TRUE/FALSE - có hiển thị thời gian không
  • message_type: 🆕 MỚI - "text" (mặc định) hoặc "image" (để gửi ảnh)

💡 Mẹo: Chỉ cần 4 cột BẮT BUỘC có ⭐, các cột khác có thể để trống!

📸 GỬI ẢNH: Đặt message_type = "image" và điền URL ảnh vào message_text
